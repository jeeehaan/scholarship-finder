# Scholarship Finder

Clone this repository

```shell
  git clone git@github.com:jeeehaan/scholarship-finder.git
```

Go to the project

```shell
  cd scholarship-finder
```

Create a Virtual Environment

```shell
  python3 -m venv .venv
  source .venv/bin/activate
```

Install Dependencies

```shell
  pip install -r requirement.txt
```

Run the project

```shell
  ./manage.py runserver
  npm run tw
```
